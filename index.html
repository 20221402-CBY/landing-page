html5 태그들을 무더보쟈!

HTML5는 단순히 "보여주는 것"을 넘어, 웹브라우저와 검색 엔진에게 데이터의 '의미'를 정확히 전달하고, 복잡한 기능을 태그 하나로 해결하려는 방향으로 진화했습니다.

1. HTML5의 주요 시맨틱 태그

웹페이지의 레이아웃을 만들 때 사용하며, 검색 엔진이 사이트 구조를 파악하는 데 아주 큰 도움을 줍니다.
태그 설명
<header> 페이지나 섹션의 머릿말 (로고,메뉴 등)
<main> 문서의 핵심 콘텐츠 (페이지당 한 번만 사용 권장)
<nav> 네비게이션 링크의 집합 (메뉴 바)
<section> 문서의 일반적인 섹션 (주제별 그룹화)
<article> 문서의 독립적인 콘텐츠 (블로그 게시물, 뉴스 기사 등)
<aside> 주요 내용 외의 사이드바 (광고, 링크 등)
<footer> 페이지나 섹션의 바닥글 (저작권, 연락처 등)

2. 텍스트 및 콘텐츠 태그

가장 기본이 되는 글자 관련 태그들입니다.
용도 설명
제목 <h1>부터 <h6>까지 사용하며 숫자가 작을수록 중요도가 높습니다.
본문 <p>(단락), <br>(줄바꿈), <span>(줄 내부 범위)
강조 <strong>(중요한 의미의 굵게), <em>(강조하는 기울임)
목록 <ul>+<li>:순서 없는 목록 (점), <ol>+<li>:순서 있는 목록 (숫자)
링크 <a href="URL"> (태그를 사용하여 다른 페이지나 웹사이트로 연결)

3. 멀티미디어 태그

HTML5에서 가장 강력해진 부분으로, 별도의 플러그인 없이 미디어를 재생할 수 있습니다.
태그 설명
<img> 이미지 삽입
<video> 동영상 재생 (controls 속성으로 재생 바 표시 가능)
<audio> 오디오 재생
<canvas> 자바스크립트를 이용해 그래픽이나 애니메이션을 그릴 때 사용
<svg> 벡터 그래픽을 정의하는 데 사용

4. 폼(Form) 및 입력 태그

사용자로부터 데이터를 입력받을 때 사용합니다. HTML5에서는 입력 타입이 훨씬 다양해졌어요.
태그 설명
<input> 다양한 타입을 가집니다.(type="text", password, email, number 등)
<datalist> 입력창에 추천 검색어 리스트를 보여줍니다.
<button> 클릭 가능한 버튼을 생성합니다.

5. 표(Table) 태그

데이터를 표 형태로 나타낼 때 사용합니다.
태그 설명
<table> 데이터를 표 형태로 나타냅니다.
<tr> 표의 행을 정의합니다.
<th> 표의 헤더 셀을 정의합니다.
<td> 표의 데이터 셀을 정의합니다.
